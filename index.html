Temperature Conversion

This Ruby program will prompt the user for a temperature in degrees Celsius and let the user know what the corresponding temperature is in Fahrenheit. 


#Multiply by 9, then divide by 5, then add 32
def converttofahr(n)
    return n.to_f * 9 / 5 + 32
end

puts "Whats the temperature where you are in Celsius?"
temperature = gets.chomp
puts "So it is #{temperature}?"
puts "Converting to Fahrenheit..."

puts converttofahr(temperature)
puts "Success!"
puts "Goodbye"

Foobar

This Ruby program is an adaption of a classic technical interview problem which displays sequences of the foobar pattern. 


puts "Let's play foobar!"
puts "Enter a number:"
n = gets.chomp.to_i
def foobar(n)
    (1..n).each do |i|
        if i % 3 == 0 && i % 5 == 0
            puts "FOOBAR"
        elsif i % 3 == 0 
            puts "FOO"
        elsif i % 5 == 0
            puts "BAR"
        else
            puts i
        end    
    end
end

foobar(n)
puts "Goodbye"
